简单的markdown入门

1. 粗体斜体

**加粗**
*斜体*

2. 段落与换行

1）不同段落间有空行
2）段内换行：上一行结尾插入两空格后回车

3. 列表

有序列表：数字符号+英文句号+空格+列表内容
无序列表：* / - / + （主要使用减号-）

  • 星号表示
  • 减号表示
  • 加号表示
    嵌套列表表示如下（有序和无序可互相嵌套）：
  • 第一层列表
    TAB + 第二层列表
    TAB + TAB + 第三层列表
  1. 嵌套列表
    • 无序列表
      1. 第三层列表

4. 分割线

分割线由3个以上的 * / - / _ 来标记

5. 图片

！图片替代文字
1)图片替代文字在图片无法显示时有用，正常可以为空
2)图片地址可以为网络或本地地址，本地支持相对和绝对路径。

13.表格

表格语法如下：

|表头1|表头2|表头3|
|---- |----|----|
|内容1|内容2|内容3|
|内容1|内容2|内容3|

语法说明如下：

1)单元格间用|来分隔。

2)单元格和|间的空格会被移除

3)表头与其他行用-来分隔

4)表格对其格式如下

- 左对齐（默认）:----
- 右对齐              ----：
- 居中对齐           :----:

5）块级元素（代码块、引用区块）无法插入表格，可插入链接等标记

示例

左对齐 居中对齐 右对齐
1 2 3
张三 李四 王五

17. 排版

对比

17.1 空格

​ 需要空格的情况

  • 中文和英文之间加空格

  • 中文/英文和数字之间加空格，注意部分编辑器会自动添加。

  • 英文标点符号与后面字符之间需要加空格，与前面的不需要

    More ways to shop: Visit an Apple Store, call 1-800-MY-APPLE.

  • 中文、英文中使用>（半角）标识路径时，两边都需要加空格

    Settings > General > Reset > Erase all Content and Settings

    不需要的情况

  • 中文标点符号和数字/中文/英文之间不需要添加空格

  • 数字和百分号之不需要添加空格

  • 数字和单位符号之间不需要添加空格

  • 英文和数字组合

  • 当/(半角)表示“或”、“路径”时，前后均不加空格

  • 货币符号后不加空格

  • 负号后不加空格

17.2全角和半角

全角：中文标点符号，占两个字节。 （，。；：！#）

半角：英文标点符号和数字，占一个字节。 （,.:;!#）
