转义字符

\a	响铃（BEL）
\b	退格（BS），将当前位置移到前一列

\r	回车（CR），移到本行开头
\t	水平制表（HT）
\v	垂直制表（VT）
\0	空字符（NULL）
\ddd	1到3位八进制所代表的任意字符
\xhh	1到2位十六进制所代表的任意字符
\\、\'、\"、\?	其他转义用

扩展函数

sort()函数：永久性改变列表顺序

用法：列表名.sort()

a=[2,3,1,4]
a.sort()
print(a)
#a=[1,2,3,4]

sorted()函数：短暂的对列表排序

用法：sorted(列表名)

a=[2,3,1,4]
print(sorted(a))
print(a)
#a=[1,2,3,4]
#a=[2,3,1,4]

reverse()函数：使列表逆序

用法：列表名.reverse()

a=[2,3,1,4,3]
a.reverse()
print(a)
#a=[3,4,1,3,2]

len()函数：得到列表或字典长度

用法：len(列表名)

a=[1,2,3]
print(len(a))
#3

insert()函数：在指定位置插入元素

列表名.insert(索引，值)

a=[1,2,4]
a.insert(2,3)
#a=[1,2,3,4]

pop()函数：删除元素并返回删除值

用法：列表名.pop(索引)

a=[1,2,3,4]
b=a.pop(1)
#b=2

remove()函数：用元素进行检索删除，返回None值

用法：列表名.remove(元素值)

统计相关函数：min()、max()、sum()

用法：函数名(列表名)

list()&range()：创建数字列表

循环结构

for...in..

in 后可接列表、字典、字符串。

遍历字符串

for i in 'abc':
    print(i)
# a b c

遍历列表

for i in [1,2,3,4]:
    print(i)

遍历字典

1. 遍历字典键和值
d={1:'a',2:'b',3:'c'}
for a,b in d.items():
    print('key:'+str(a)+'\value:'+str(b))
# items()返回字典的(键，值)元组分别赋给a,b
2. 遍历字典的键
d={1:'a',2:'b',3:'c'}
#method 1
for i in d:
#method 2
for i in d.keys():
    print(i)
3. 遍历字典的值
d={1:'a',2:'b',3:'c'}
for i in d.values():
    print(i)

range()函数

range(a,b,c)
#a:计数从a开始，不填默认为0；
#b:计数到b结束，但不包括b；
#c：计数间隔，不填时默认为1；
1. 整数序列：取头不取尾
for i in range(1,3):
    print(i)
#1、2
2. 带间隔数
for i in range(1,10,3):
	print(i)
# 1、4、7
3. 次数
for i in range(x):
	print i  #从0到x-1,执行x次

random 模块

import random

random.randint(x,y)
#随机产生[x,y]的整数

random.uniform(x,y)
#随机产生[x,y]浮点数

random.choice(x)
#从x序列中随机选取一个元素，如random.choice('Python')

random.randrange(x,y,z)
#生成[x,y]的间隔为z的随机整数

random.shuffle(x)
#打乱序列x中的元素顺序，如random.shuffle([1,2,3,4,5])

random.sample(x,y)
#从x字符串中随机选取y个字符

函数

定义函数

def 函数名(argu1,argu2....):
	函数体
	return语句

默认参数

#默认参数必须放在位置参数之后，调用时若没有传递参数则为默认值
def func(a,b,c=0):
    print(a+b+c)
func(1,2)
func(1,2,3)
#3
#6

不定长参数

#当传入参数数量不确定时，可以在参数名前加*，此时数据类型为元组（tuple）。
def func(*argu):
	return argu
print(func(1,2,3))
#1 2 3

#若不定长参数后又默认参数，要修改默认参数，需注明默认参数
def func(*argu,a='0'):
    ...
func(1,2,3,a='3')

以print()为例

print(*objects,sep=' ',end='\n',file=sys.stdout,flush=False)
#*object说明函数可以传递任意数量参数
#sep控制多个值之间的分隔符，默认空格，如sep='+'则会加上+号
#end控制打印结果的结尾，默认是换行

return

#可以以元组形式返回多个值,如
return 1,2,3
